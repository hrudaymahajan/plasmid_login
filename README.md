# Modern Login & Register UI

A clean and modern **Login & Register Authentication UI** built using HTML, CSS, and JavaScript.
This project features a glassmorphism design, smooth transitions, and form validation.

---

##  Live Demo

https://your-username.github.io/login-register-ui/

---

##  Features

* Toggle between Login & Register forms
* Modern Glassmorphism UI
* Show/Hide Password
* Form Validation (Email, Password, Confirm Password)
* Fully Responsive Design
* Smooth animations and transitions

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Flexbox + Glassmorphism)
* JavaScript (Vanilla JS)

## ⚙️ Setup Instructions

1. Clone the repository:

```
git clone https://github.com/your-username/login-register-ui.git
```

2. Navigate to the project folder:

```
cd login-register-ui
```

3. Open `index.html` in your browser

---

## 🌐 Deployment (GitHub Pages)

1. Go to repository **Settings**
2. Click on **Pages**
3. Select:

   * Branch → `main`
4. Click **Save**

Your site will be live at:

```
https://your-username.github.io/login-register-ui/
```
